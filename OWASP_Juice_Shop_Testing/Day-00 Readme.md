# Web Security Analysis Series (15 Days)

This repository contains my **15-Day Web Security Analysis Series**, where I performed hands-on testing on **OWASP Juice Shop** to identify and analyze real-world web application vulnerabilities.

---

## Overview

The goal of this series was to gain practical experience in identifying, exploiting, and understanding common web vulnerabilities using industry-standard tools.

Each day focuses on a specific vulnerability with:
- Step-by-step analysis  
- Request/response manipulation  
- Screenshots and explanations  
- Impact and mitigation strategies  

---

## Tools Used

- Burp Suite  
- Postman  
- Web Browser Developer Tools  

---

## Vulnerabilities Covered

- Cross-Site Scripting (XSS) – Reflected & Stored  
- SQL Injection  
- Broken Access Control  
- Insecure Direct Object Reference (IDOR)  
- Information Disclosure  
- Privilege Escalation  
- API Security Issues  
- Business Logic Flaws  

---

## Key Highlights

- Hands-on testing using **Burp Suite and Postman**  
- API-level vulnerability testing and manipulation  
- Input validation bypass techniques (e.g., encoding tricks)  
- Realistic attack scenarios and exploitation methods  
- Structured documentation for each finding  

---

## Learning Outcomes

- Improved understanding of **OWASP Top 10 vulnerabilities**  
- Practical experience in **web application security testing**  
- Ability to analyze and manipulate **HTTP requests and APIs**  
- Strong foundation for **SOC Analyst and Application Security roles**
